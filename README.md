Latest News Reader (Simple Pie Based) Laravel Package

1 - Require

	composer require heymowski/latest-news-reader

2 - Migrate DB

	Artisan Migrate

3 - Add News Source

	Artisan LNR-Sources:AddNewsSource

4 - Load Feeds to db

	Artisan LNR-Items:ReadAll
